Simple script to create dictionaries for pentesting.


# Use

Clone repo
```
git clone https://github.com/TobiasCoding/create_dictionary.git
```

Go to repo
```
cd create_dictionary
```

Execute the script
```
python3 create_passwords_list.py
```

Check output
```bash
tail -f passwords.pst
```
